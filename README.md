# Development

* Java SDK 1.8
* Eclipse Neon
* Maven 3.3.9
* Spring framework
* JOOQ
* MySQL database connection

# Quick start

In the root project folder:

```bash
$ mvn dependency:tree
$ mvn spring-boot:run
```

Open [http://localhost:8080](http://localhost:8080).

# Import to Eclipse

In the root project folder:

```bash
$ mvn eclipse:eclipse
```

* Open Eclipse -> Import Maven project.
* Run application

Open [http://localhost:8080](http://localhost:8080).
